# 【记小栈博客】 小程序版

一个精美漂亮的小程序博客。

<img src="https://static.liayal.com/images/Ji-Blog-weapp.jpg" />

## 项目描述

- TypeScript 开发
- 使用 Gulp 构建 
- 独立的UI设计

## 项目运行

#### 安装依赖

```bash
$ npm install
```

#### 启动

```bash
$ npm start
```
